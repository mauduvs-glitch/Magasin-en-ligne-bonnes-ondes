# Magasin-en-ligne-bonnes-ondes
Bonne ondes, le site en ligne pour vos achats musique

Voici une page Web réalisée dans le cadre d’un devoir de HTML/CSS en cours de documentation audiovisuelle.

Pour alimenter un site web nommé Bonnes-Ondes, la page présente une sélection d’instruments de musique (guitares, batteries, claviers) ainsi qu’une bibliographie musicale.

La bibliographie a été exportée au format HTML depuis Zotero et intégrée à la page avec conservation des métadonnées COinS.

Le site est publié avec GitHub Pages.
 
<href1>Outils</href1>
    HTML5 comme support / CSS3 pour la présentation
    les identifiants d'autorité pour la désambiguïsation
    les microdatas pour les liens sémantiques
    avec schema.org pour la structuratition
    Le dublin core pour les métadonnées de la page au sein de la balise <head> (POSH : Plain Old Semantic Metadata)
    Le plugin OSDS pour la lecture augmentée
    Le protocole Z39.88 avec COinS (context object in span) pour la compatibilité bibliographique avec Zotero via la balise <span>
    Le validateur W3C pour vérifier la qualité du HTML5
    Le validateur schema.org pour vérifier la qualité des relations sémantiques intégrées dans la page.
